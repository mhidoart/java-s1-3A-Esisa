Introduction au langage HTML

1. Introduction

Le langage HTML (HyperText Markup language) est un langage de marquage, ou encore un langage de balisage qui a été créé en 1991 pour accompagner le Web.
Dans cette introduction nous alons présenter les balises de base ainsi que les bonnes pratiques HTML.
2. Historique du Web
L'historique du langage HTML remonte à l'année 1969 :
GML :
Premier langage de balisage créé en 1969 dans les locaux d'IBM par ses 3 concepteurs : Goldfarb, Mosher et Lorie.
SGML :
Il s'agit d'une standardisation du langage GML, en 1974, pour désigner Standard Generalized Markup Language.
WWW :
Signifiant World Wide Web ayant été initié en 1989 par Tim Berners-Lee.
HTML :
HyperText Markup Language, issu du SGML et présenté la première fois par Tim Berners-Lee en 1991, pour être proposé ensuite comme standard de présentation des documents sur le Web. Une grande liste de balises a été alors créée pour fournir différents styles d'affichage de contenu.
